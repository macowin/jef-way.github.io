Inputting+ - Android 上的「Ctrl+Z」

Veröffentlicht am 2015-11-23

和心上人聊天的时候，我们总是斟酌用词、语气严谨，生怕说错什么让对方误解。所以在说话时，我们常常会经历「输入、删除、输入、删除」的往复过程，可是翻来覆去还是觉得第一次说的比较靠谱，于是只好再重新输入一遍。然而当我们发送完这句话，这时候往往已经过去了好几分钟，对方是不是以为我对 TA 爱答不理呢？如果手机上能够做到桌面系统上的「撤销」功能那不就方便多了。

少数派曾经为大家介绍过 Type Machine，它能够记录所有你输入过的文字，然而它也仅仅是记录一段输入完整的文字，不能记录输入的历史动作，想要用它实现「撤销」和「重做」确实有些捉襟见肘。而 Inputting+ 这款应用不仅能够实现 Type Machine 的所有功能，还高精度记录了我们所有的录入动作，从此，我们打字也可以吃后悔药了。

记录一字一符

Inputting+ 的主要目的就是做到 PC 或 MAC 上的「撤销」和「重做」功能，这种功能的原理无非就是系统或应用在后台记录你输入过的东西，然后再需要的时候一次一次恢复回来。

Inputting+ 的运作方式非常简单，当我们点击输入框，Inputting+ 的悬浮球会与输入键盘同时弹出，我们将候选词上屏的时候，Inputting+ 会在后台记录这次上屏内容，下一次再有内容上屏的时候，如果同一个输入框中有内容，那么 Inputting+ 会认为第二次上屏内容为增量输入，从而将这次输入的文字作为撤销的一个步骤。

当我们需要执行撤销或重做动作，只需要点击悬浮球，在弹出框中进行相应的动作，完成后点击返回键，Inputting+ 会通过剪切板自动复制粘贴，快速省事儿。

除了撤销和重做，Inputting+ 还可以进行文字查找，逐个删除或者替换文字的时候非常有用。

放心一步一路

Inputting+ 同样可以实现 Type Machine 的输入时间线功能。Inputting+ 中的输入时间线功能需要付费解锁，支持 Google Play 付款或支付宝支付。支付宝支付结束后点击我已购买即可解锁。开发不易，请大家支持正版。

解锁输入时间线功能后，再次进入 Inputting+ 会直接看到时间线，我们曾经输入过的内容以卡片形式按照时间顺序排列，并且每一个卡片都单独记录了上屏内容，可以直接在卡片中进行撤销或重做，重复输入内容方便至极。

值得一提的是，为了保护隐私，Inputting+ 在输入时间线界面是不允许截图的。无论是使用系统快捷键，或是豌豆荚之类的手机助手，亦或是通过 AirMirro，我们都无法直接获取屏幕快照。Inputting+ 的这点设计值得称赞，可见开发者的用心良苦。

除此之外，Inputting+ 可以与「Clip Stack 剪纸堆」联动，这是一款剪切板历史记录应用，两者相互配合，对于某些重复性较高的文字录入工作非常有用。

撤销和重做功能虽然不起眼，在桌面系统里实现起来也是易如反掌，而在手机端却是我们非常需要的功能。Inputting+ 的出现让我们可以放心的去「后悔」。如果你是一个文字工作者，或者是一位喜欢用手机码字的人，那么 Inputting+ 一定是你需要的工具。

Ice Box - 把耗电应用都冻起来

Veröffentlicht am 2015-11-20

Android 系统越来越流畅，用户体验在慢慢改善，但是耗电却始终是个棘手的问题。虽然大部分应用能够「守本分」，但是总有那么一小撮应用，即使退出了之后还「阴魂不散」，偷偷的在后台蚕食你手机宝贵的电量。眼看 Android 6.0 的 Doze 模式依旧遥远，是时候找一个「冰箱」把那些耗电大户「冻」起来了。

而 Ice Box 就是这样一个冰箱，不过这款应用需要 Root 权限。

不乖的，把它们放进去冻起来

总有一些应用，不会经常用到，隔三岔五的又需要打开看一下。卸载了麻烦，不卸载又老是在后台自启动，躲在后台蚕食本来就不多的电量。像这种「电老鼠」，总是让人头疼。

这个时候你就需要这样一台「冰箱」：Ice Box 来帮助你。打开它可以看见一段简单的说明，点击左边「编辑」按钮添加需要「冻结」的应用，中间的「雪花」按钮就可以冻结这些应用，另外，在手机锁屏后也会自动冻结这些应用。

而在设置界面，你可以设置锁屏后多久冻结应用，或者将
系统应用
加入列表（冻结有风险，设置需谨慎），如果你不想用了，就选择清空冻结列表。

而在日常使用中，在设置好冻结列表之后，你就忽略掉它了，锁屏后会自动冻结耗电应用（冻结的应用名字面前会有雪花符号），被冻结的耗电应用不会出现在桌面。等你想要使用这些应用的时候，打开 Ice Box 就可以看见它们。

不是冷藏，是冰冻

相信熟悉 Android 应该都和我一样猜到了，这款应用「冻结」应用的方式不是像绿色守护那样仅仅停止运行，它是「停用」了整个应用，所以桌面和应用抽屉都看不见应用。在这种情况下，应用等同于未安装状态，不可能有一丝耗电。而在使用应用时，则需要在 Ice Box 里面打开。

Pintasking - 让手机也如「 Alt + Tab 」般得心应手

Veröffentlicht am 2015-11-15

得益于硬件水平的不断提升，我们的移动设备似乎已能胜任大多数情况下的多任务处理工作。但内存大并不代表体验佳，尽管我们几乎能在任何时候便捷地呼出多任务管理界面，当后台任务堆积过多时，不手动进行一番清理我们依旧很难在诸多任务中进行快速定位和切换。

如何才能专注于某几项特定的任务而不受多余的干扰，Pintasking 或许能给我们一个完美的解决方案。

对「悬浮窗」的又一次「再创作」

我曾经向 @Jeffery.Du 推荐过一款名叫 BuMP Music Player 的音乐播放器，这款播放器利用了悬浮窗的特性，在保证应用自身体积轻量、小巧的同时还能在最大程度上降低我们在阅读、工作以及浏览网页时的注意力分散状况（类似的应用还有 Link Bubble 以及 Flynx ）。

Pintasking 则更加「会玩儿」一点，利用悬浮窗，Pintasking 可以将我们的屏幕边缘变成两条灵活的「任务栏」。

在系统设置的「辅助功能」（ Lollipop 5.1 中又叫「无障碍」）中开启 Pintasking 服务后（已 root 用户可安装内置的 System Helper 来获得最佳体验），Pintasking 就会开始在后台保持运行；
在使用过程中，我们只用把那些需要集中注意力解决的任务通过特定操作「钉」（ pin ）在屏幕边缘即可，次要任务则可以忽略不管
。

这里的「特定的操作」有三种：

  • 点击下拉通知栏中常驻的「 Tap to pin 」
  • 在设定好的区域（如状态栏中部）滑动触发（ SwipePad ）
  • 长按 Home 键选择「 Pin this task 」（在带有虚拟屏幕按键的设备上为按住 Home 键上滑）

以具体的使用场景为例：当我在 Feedly 看到一条值得翻译过来并推送至微博的消息时，只需在 Feedly 、需要用到的翻译工具以及微博这三个应用中把它们「钉」到屏幕边缘，然后便可以通过悬浮窗在这三个任务间快速切换。

这样的多任务处理方式的好处是显而易见的――一方面，将需要处理的任务以悬浮窗的形式放在屏幕一侧，我们就无须在系统最近任务中进行查找和定位，节省时间、提高效率；另一方面，由于悬浮窗默认是全局置顶的，即便我们的注意力不得不暂时被其他应用分散和打扰、亦或是熄屏离开后重新点亮屏幕，悬浮窗的存在都能时刻提醒我们还有待处理的任务。

用细节定义完美

在上手一段时间后，我最终没忍住选择了内购解锁 Pintasking Pro ，专业版强大而细致的设置功能让 Pintasking 成为了我手机里又一款不可或缺的效率工具。下面就简单罗列一些：

  • 自定义悬浮窗大小以及悬浮窗被屏幕边缘遮挡的范围

  • 自定义悬浮窗任务的自动消失条件
    。可以设置「熄屏后自动关闭所有悬浮窗任务」，也可以勾选「只允许手动关闭悬浮窗任务」

  • 设置悬浮窗任务自动停靠区域
    。可禁用上、下边缘以避免误操作；

尽管免费版已经完全能够满足我们的日常使用要求，花 3.99 美元永久解锁之后通过各方面调教可以让 Pintasking 用起来更加得心应手，何乐而不为？

浏览器插件：「 Link Bubble 什么的通通下岗吧。」

不得不提的是，除了多任务处理，Pintasking 还提供了一个叫做「浏览器扩展」（ Browser Extension ）的功能。

该功能看似与 Link Bubble 、Flynx 相类似，实则有很大的区别―― Link Bubble 与 Flynx 可以看作是独立的浏览器，
而 Pintasking 则需要一款其他的浏览器作为其扩展对象
：当我们点击第三方应用中的链接时依旧需要其他浏览器来打开这个链接，随后 Pintasking 会在浏览器右下方显示一个工具栏，点击工具栏上的图钉按钮，这个网页就会被「 Pin 」到悬浮窗并自动切换到第三方应用中方便我们继续浏览。

我们也可以这个「浏览器扩展」功能看作是一个「网页收集器」
，将我们在使用其它应用的过程中所打开的网页集中起来在后台打开以保证体验的流畅性。这一点与 Link Bubble 有异曲同工之处。

写作的时候需要什么？这个问题在各个编辑器文章中已经讨论了无数遍。我们之所以一再推荐 Markdown 也是基于这样的讨论，Markdown 通过简单的标记语言屏蔽了写作过程中可能影响思考进程的几乎一切因素，形成了一个只需要考虑文字创作本身，而不会被排版之类的「噪音」影响的体验。

Monospace 可以通过长按选择区域来进行格式的简单设置以及剪切复制等编辑操作。值得注意的是，这里允许设置的六个格式都可以直接对应 Markdown 的语法，所以看起来像「字体缩放」的两个按钮经过实测其实是正文与某级标题切换，这个设计不太直观。

Monospace 支持导出 Markdown 格式或纯文本，右侧即为 Markdown 的导出结果，在锤子便签上预览毫无压力。同时 Monospace 支持绑定 Dropbox，与你的其他设备同步也不再是问题。

现在我已经将 Monospace Writer 作为日常的便签与文段管理工具使用，极短的开启时间，方便的标签管理模式，非常适合进行素材整理，这也是之前我使用的锤子便签所欠缺的地方。写完的小文段就删掉「#草稿」加上「#写完」，不过你也看到了，现在一个「#写完」也没有出现过……这就不是工具能够解决的问题了。需要提醒的是，Monospace Writer 并没有对中文进行完整适配，在一定情况下会出现中文字号混乱的现象，但并不影响日常使用。

平时我们上网查阅资料、阅读文章等等都有收藏或稍后阅读的需求。而随着人们电脑、手机、平板等各种设备越来越多，当然也希望这些文章收藏/网页书签能在任何设备上都能自动同步了。

虽然现在很多浏览器都有收藏夹同步功能，但如果将各类待读文章与网址收藏夹混杂在一起似乎并不好管理，整理起来很伤脑筋，而且也不能方便地搜索收藏过的内容。Raindrop.io 是一个跨桌面和移动平台的免费
云端网络书签服务
，拥有精致的设计以及方便的收藏保存、分类整理、搜索以及书签同步等功能……

收藏网页的利器、文章稍后阅读清单 - 你喜爱的网站，触手可及！

传统的浏览器收藏夹功能仅仅提保存了网页的标题和网址，收藏一时爽，但项目多起来后想要查找或整理就让人死的心都有了。我们认为，一款
强大的网络书签工具
，应该是一个可以在不同设备之前自动同步稍后阅读 (网页/文章) 列表，不仅要可以轻松分类整理，还要可以快速搜索查找内容！

Raindrop.io
做到了想要的一切，简洁、美观，跨平台提供 Android / iOS / Mac 平台客户端、网页版以及各大浏览器的插件扩展。你的所有书签都会保存在云端，无论你使用何种设备，都能非常方便地保存、整理以及获取你的书签收藏！！即使在他人的电脑上也能通过网页版进行访问。

归类整理、查找搜索、分享书签收藏列表以及协同管理

归类整理与查找搜索：
Raindrop.io 可以给你的收藏的网页添加描述或 Tag 标签，或创建文件夹 (Collection) 进行整理归类，可以帮助你将不同的工作参考资料、有趣的文章、学习资料、稍后待读、软件教程等分门别类整理好。而且当你需要查找相关的书签收藏时，还能通过关键字直接进行搜索，这一点比起浏览器的收藏功能实在是高效太多了。

分享收藏集：
当你收集整理了一些不错的网页清单 (Collection) 时，譬如「异次元上最值得阅读的文章列表」、「全球最佳的设计类网站」等，还可以通过 Raindrop.io 将这个书签列表分享给好基友们。这比起以前要一个一个复制网址再通过 QQ、微信发给别人简直不能更方便，而且你整理好的列表也能得到更系统化的呈现，更易于传播交流。

协同管理书签收藏：
你还可以邀请朋友、同事共同管理某个收藏集里的书签，这对于一些需要协作共同查找收集资料的朋友来说会非常实用！毕竟一直以来书签都是“私人物品”，与他人分享收藏夹里的好网站好文章都已困难重重，更别说与别人一起「共用」收藏夹了。作为互联网新一代的书签工具，Raindrop.io 在书签的分享与协同管理上的想象力空间会非常大，应用好这个特性相信一定能帮上不少忙。

更多丰富的特性：

Raindrop 还有很多特性，你不仅可以收藏网页，还能收藏图片/视频；可以给收藏夹生成私人或公共的 RSS 订阅地址；支持智能识别文章内容格式化排版；支持从 Chrome、Firefox、Opera、Safari 等主流浏览器以及 Delicious、Instapaper、Pocket、Readability 等服务导入书签；开通付费高级功能后，还可以通过 Dropbox 备份书签、支持多层级文件夹等。

另外它还提供了一个「新标签页插件」(快速拨号)，可以直接替代 Chrome 或 Firefox 的默认新建标签页，你只需要把书签加入到「最喜爱的网站」这个分类里，它们就会出现在新建标签页中了，这样就能更好地替换掉浏览器自身的书签功能了。

总结：

Raindrop.io 拥有完善强大的书签保存、分类与管理功能，提供多种平台客户端、浏览器扩展，全面覆盖手，所有书签云端自动同步，还能与他人共享与协作，基本上能想到的书签功能它都已具备，堪称收藏神器了。

而且设计精致的界面、合理的布局，以及完全免费的使用也让 Raindrop 加分不少！可以说，Raindrop.io 也是新一代书签服务的诠释者，非常适合有大量收藏资源以及喜欢整理有序的用户。作为跨平台的应用，Raindrop 无论在 iOS、Android 还是电脑浏览器上都表现出色……
