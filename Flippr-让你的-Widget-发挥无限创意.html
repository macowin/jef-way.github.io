Flippr - 让你的 Widget 发挥无限创意

作为 Android 平台引以为傲的桌面元素，Widget 的方便之处自然不必多言。桌面放个天气小部件，随时就能看到天气情况；放个备忘录，打开桌面就可以起到提醒的作用。而 Widget 最棒的一点还是它的互动性，不仅可以控制音乐播放器，快速打开应用里面的特定功能，你甚至可以在桌面放一个计算器，需要时随时计算而不用打开应用。

Widget 的存在大大地减少了我们的操作，削弱了点开应用的必要性。然而如此好用的东西，只能在桌面使用岂不可惜。

J 君曾经推荐过一款能够将 Widget 小部件放进下拉状态栏的应用 Snap ，也曾经推荐过一款悬浮音乐播放器 BuMP Music Player，前者是为了节省桌面空间，后者是为了在一边阅读一边听音乐时尽量保持少的操作。而 flippr 这款应用可以轻松的代替这它们，因为它能让你的 Widget 小部件做到「随手可得」。

无论何时，一划即得

Snap 将 Widget 放在下拉菜单栏里面，已经可以做到在任何界面使用 Widget 了，但是下拉菜单栏始终是一个让大屏手机用户苦恼的动作。flippr 则更进一步，将 Widget 改为由一个悬浮圆滑动呼出，效果颇为炫酷。

呼出后的 Widget 小部件依旧可以互动，点击小部件以外的位置则会自动收回，操作流畅毫无违和感，就像在屏幕侧面多出了一个 Widget 工具栏一样。

无论是在阅读还是在聊天，甚至是在玩游戏的时候，你都可以随时呼出你的 Widget 小部件进行类似切歌、查看清单、切换应用等操作。

这下你明白为什么 flippr 可以替代 BuMP 了吧：只要你常用的音乐播放器有 Widget 小部件，你就可以借由 flippr 获得接近 BuMP 的效果。

完善定制，随心所欲

flippr 的用户指引非常的详细，实际上这款应用的操作并不复杂，只要记住：按住圆点上下滑动选择分组，选定分组左右滑动选定。

应用的分组有两种，Widget 和 Shortcut，后者类似于快速打开应用，个人并不常用。命名分组后就可以添加 Widget 了，添加之后可以点击齿轮按钮将 Widget 大小、背景调整到合适的样子。

应用设置里面提供了丰富的选择，你可以在 Blacklist 里面选择在选定应用隐藏 flippr，也可以调整圆点的位置和大小。不过免费版不支持自定义圆点的颜色和呼出 Widget 的动画效果。

自从发现这款应用后，J 君就删除了 Snap，毕竟随手一滑和下拉状态栏的操作难度差距显而易见。这款应用提供了一个优质的 Widget 工具，至于能用它玩出什么花样，还要看你攒了多少有趣的 Widget。
