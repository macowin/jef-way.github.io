Android Studio - 谷歌官方安卓 APP 应用开发工具

发表于 2016-04-12   |  

在 Google I/O 大会上，谷歌推出了自家全新的官方安卓软件集成开发工具 Android Studio，当时引起了现场开发者们的一片欢呼,这是 Google 基于 IntelliJ IDEA (一款优秀的 Java 集成开发工具) 修改而来。

谷歌称 Android Studio 将会让开发者更快、更具生产力，它完全免费，跨平台支持 Win / Mac / Linux，是替代 Eclipse 的最佳安卓开发工具。现在移动 APP 开发正处于热火朝天的阶段，如果你有兴趣研究学习一下Android 软件应用的开发，那么试试 Android Studio 可能会有不少收获哦……

Google 官方 Android 应用开发工具 IDE

Android Studio 能让应用开发更简单，它相比以前 Eclipse 的开发工具最大特点就是支持实时渲染（所见即所得），开发者可以在编写程序的同时可以实时看到自己的应用在不同尺寸屏幕中的样子，非常的方便！相信这是无数开发者所一直期待的。

同时 Android Studio 还解决了语言问题，多语言版本、支持翻译都让开发者更方便地开发适应全球的应用。而且 Android Studio 相比 Eclipse 速度和性能上的表现要优秀得多，相信用过 Eclipse 和 IDEA 的同学一定会深有体会。而且，新版本的 Android Studio 也声称提升了三倍的速度和性能。

Android Studio 的一些特色

即时运行

按设备或仿真器上运行的代码和资源更改您的应用程序，能马上看到它的变化，即时运行大大加快您的编辑，编译和运行周期，保持你“流畅”感觉

智能代码编辑器

编写更好的代码，运行速度更快，并具有智能代码编辑器，可以帮助你的每一步更富有成效。

Android的工作室是建立在的IntelliJ和能够先进的代码完成，重构和代码分析。

快速和功能丰富的仿真器

安装并运行你的应用程序比使用物理设备速度更快，在几乎所有的Andr​​oid设备配置测试您的应用程序：Android手机，Android平板电脑，Android Wear和Android TV设备。

新的Andr​​oid模拟器2.0比以往任何时候都更快，允许你动态调整仿真器和访问一套传感器控制。

强大而灵活的构建系统

轻松地配置您的项目，包括代码库，并​​从单个项目中的多个变种版本。

随着摇篮，Android的Studio提供高性能的构建自动化，强大的依赖管理，并定制构建配置。

适配所有Android设备

针对多种外形尺寸与一个单一的项目，以轻松共享不同版本的应用程序之间的代码。

Android的Studio提供了一个统一的环境中开发Android手机，平板电脑，Android Wear，Android的电视和Android应用程序自动。

代码模板和GitHub上的整合

开始的模式，如导航抽屉，查看传呼机，或从GitHub进口谷歌代码示例代码模板项目。

Android的Studio的项目向导使它比以往更容易在新项目中添加代码。

  • Instant Run 即时运行代码
  • Android Emulator 新的安卓模拟器
  • 云端测试实验室集成
  • 应用索引代码生成与测试
  • GPU调试器预览
  • IntelliJ 15 更新
